# AutoRepo
## 🪧 À propos
AutoRepo est une petite application JavaFX qui permet de créer automatiquement un dépôt GitHub via l’API, puis de le cloner dans un dossier choisi.
Le but : gagner du temps et éviter les manipulations répétitives.

## 📦 Prérequis
Java 17+

Git installé

Maven (si tu veux lancer avec mvn)

!!! Un token GitHub (non stocké dans le code) !!!

## 🚀 Installation
bash
git clone https://github.com/QuentinLeger/Creation_dun_repo_Github.git
cd Creation_dun_repo_Github
## 🛠️ Utilisation
### Lancer l’application
Méthode simple :

bash
mvn javafx:run
Méthode alternative (si tu génères un .jar) :

bash
java -jar AutoRepo.jar
### Fonctionnalités
Création automatique d’un dépôt GitHub

Clonage dans un dossier choisi

Génération d’un README et d’un .gitignore

Initialisation Git (init, add, commit)
